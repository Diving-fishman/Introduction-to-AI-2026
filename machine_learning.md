## 一、🧠 机器学习基础概念

### 知识点 1：什么是机器学习？

**定义**：计算机利用**经验**（数据）来改善**性能**指标的过程。

```
输入：数据（经验）
过程：学习算法从数据中发现规律
输出：一个模型（可以用来做预测）
```

**一个核心公式**：
```text
学习算法 + 训练数据 → 模型 → 预测新数据
```

### 知识点 2：基本术语

| 术语 | 英文 | 含义 | 比喻 🌰 |
| :---- | :---- | :---- | :------ |
| **样本** | Sample / Instance | 一条数据记录 | 一个学生的信息 |
| **特征** | Feature | 样本的某个属性 | 年龄、身高、成绩 |
| **特征向量** | Feature Vector | 所有特征组成的向量 | (年龄, 身高, 成绩) |
| **标签** | Label | 我们要预测的目标值 | 是否被录取 ✅❌ |
| **数据集** | Data Set | 所有样本的集合 | 全班学生的数据 |
| **训练集** | Training Set | 用来训练模型的数据 | 过去学生的数据 |
| **测试集** | Test Set | 用来评估模型的数据 | 新学生的数据 |

### 知识点 3：几个重要的假设

| 假设 | 含义 |
| :---- | :---- |
| **独立同分布（IID）** 📊 | 训练数据和测试数据来自同一分布，且相互独立 |
| **数据充分性** ✅ | 数据量要足够大，才能学到可靠规律 |

## 二、📐 机器学习的核心框架

### 知识点 4：模型、假设空间与学习

**模型** = 从输入到输出的映射函数：

```text
线性回归：y = wx + b              ← 参数 w 和 b 控制模型行为
神经网络：y = σ(W·x + b)          ← 参数 W 和 b
决策树：一系列 if-then 规则        ← 树结构参数
```

**假设空间（Hypothesis Space）**：所有可能的模型组成的集合。

```text
假设空间 H = { f(x; θ) | θ ∈ Θ }

θ 是模型参数，Θ 是所有可能的参数取值
H 中包含了所有可能的学习结果
```

> 💡 机器学习 = 在假设空间中搜索一个"最好"的模型，使其在训练数据上表现最好。

### 知识点 5：损失函数与风险

**损失函数 L(y, f(x))**：衡量模型预测值和真实值之间的差距。

| 任务 | 常用损失函数 |
| :---- | :---- |
| **回归** 📈 | 均方误差 MSE：L(y, ŷ) = (y - ŷ)² |
| **分类** 🏷️ | 交叉熵（Cross-Entropy）、0-1 损失 |

**三种风险最小化：**

```text
① 期望风险（Expected Risk）—— 理论上最优，但算不了
   R(θ) = ∫ L(y, f(x,θ)) · p(x,y) dx dy

② 经验风险（Empirical Risk）—— 用训练数据近似
   R_emp(θ) = 1/N · Σ L(yᵢ, f(xᵢ,θ))

③ 结构风险（Structural Risk）—— 加正则项防过拟合
   R_srm(θ) = R_emp(θ) + λ·Ω(θ)
```

### 知识点 6：过拟合 vs 欠拟合

```text
      欠拟合（Underfitting）         过拟合（Overfitting）
           😅                             😰
    ┌──────────────────┐       ┌──────────────────┐
    │  模型太简单       │       │  模型太复杂       │
    │  训练集也学不好    │       │  训练集完美但      │
    │  测试集更差       │       │  测试集很差        │
    └──────────────────┘       └──────────────────┘

           👍 刚刚好（Good Fit）
    ┌──────────────────┐
    │  模型复杂度适中    │
    │  训练集和测试集    │
    │  表现都很好        │
    └──────────────────┘
```

**训练过程示意图：**
```text
误差
↑
│     训练误差
│    ┌──┐
│   ┌┘  └──┐            测试误差
│  ┌┘      └──┐       ┌──┐
│ ┌┘          └──┐   ┌┘  └──┐
│ ┌┘              └───┘      └──────→ 模型复杂度
│←--- 欠拟合 ---→←--- 过拟合 ---→
```

| 问题 | 原因 | 解决方法 |
| :---- | :---- | :---- |
| **欠拟合** 😅 | 模型太简单 / 特征不够 | 增加模型复杂度、增加特征 |
| **过拟合** 😰 | 模型太复杂 / 数据太少 | 正则化、减少特征、增加数据 |

### 知识点 7：机器学习的典型流程

```text
① 收集数据 📝
② 数据预处理（清洗、归一化、特征工程） 🧹
③ 划分训练集 / 验证集 / 测试集 ✂️
④ 选择模型（线性回归、SVM、神经网络等）
⑤ 设置超参数（学习率、正则化系数等） ⚙️
⑥ 在训练集上训练模型 🏋️
⑦ 在验证集上调优 🔧
⑧ 在测试集上评估 🧪
⑨ 部署到实际应用 🚀
```

## 三、🏷️ 监督学习（Supervised Learning）

### 知识点 8：什么是监督学习？

**输入**：带标签的训练数据 {(x₁, y₁), (x₂, y₂), ..., (xₙ, yₙ)}
**任务**：学习从 x 到 y 的映射 f: X → Y

```text
             ┌── 分类（Classification）
监督学习 ────┤                y 是离散的（猫/狗、正/负）
             │
             └── 回归（Regression）
                          y 是连续的（房价、温度）
```

**常见算法：**
| 算法 | 类型 | 一句话说清 |
| :---- | :---- | :--------- |
| **线性回归** 📈 | 回归 | y = wx + b，找到最拟合数据的直线 |
| **逻辑回归** 📉 | 分类 | 用 Sigmoid 函数把输出变成 [0,1] 的**概率** |
| **决策树** 🌳 | 分类/回归 | 一堆 if-then 规则组成的树 |
| **SVM** 🏋️ | 分类 | 找到两个类别之间最宽的"间隔" |
| **朴素贝叶斯** 🧮 | 分类 | 用概率公式 P(y\|x) ∝ P(x\|y)P(y) |

## 四、🔍 无监督学习（Unsupervised Learning）

### 知识点 9：什么是无监督学习？

**输入**：只有数据 xᵢ，没有标签 yᵢ
**任务**：发现数据中隐藏的结构或模式

```text
               ┌── 聚类（Clustering）— 物以类聚
无监督学习 ────┤
               └── 降维（Dimensionality Reduction）— 简化数据
```

### 知识点 10：K-Means 聚类

**核心思想**：把数据分成 K 个簇，让**同一簇内的点尽量近，不同簇的点尽量远**。

**K-Means 的步骤：**
```text
① 随机选 K 个点作为初始"中心"
② 每个点分给离它最近的中心
③ 更新每个簇的中心（取簇内点的平均值）
④ 重复 ②③ 直到中心不再变化
```

```text
初始状态        第1轮        第3轮        收敛
  · · ·        · · ·        · · ·        · · ·
  · · ·   →    · · ·   →    · · ·   →    · · ·
 ·⊕· ·· ⊕·    ·⊕· ·· ⊕·    ·⊕· ·· ⊕·    ·⊕· ·· ⊕·
  · · ·        · · ·        · · ·        · · ·
  · · ·        · · ·        · · ·        · · ·
⊕ = 簇中心    · = 数据点    颜色 = 隶属簇
```

**常见降维方法**：**PCA**（主成分分析）📉

## 五、🔄 其他学习范式

### 知识点 11：弱监督学习（Weakly Supervised Learning）

**定义**：使用不完美监督信号的学习方法。包含三种情况：

| 类型 | 说明 | 例子 |
| :---- | :---- | :---- |
| **不完全监督** 🕳️ | 只有部分数据有标签 | 100 万张图，只有 1000 张有标签 |
| **不确切监督** ❓ | 标签不够精确 | 一张图上有猫和狗，但只标了"动物" |
| **不准确监督** ⚠️ | 标签可能有错误 | 标注人员标错了类别 |

### 知识点 12：迁移学习（Transfer Learning）🎁

**核心思想**：把在某个任务上学到的知识，应用到另一个相关任务上。

```text
传统学习：对每个新任务都要从头学起 ❌
        任务A → 模型A    任务B → 从头训练模型B

迁移学习：在任务A上学到知识，迁移到任务B ✅
        任务A → 基础模型 → 微调 → 任务B的模型
```

> 💡 **例子**：用 ImageNet（1000 类图片分类）训练好的 CNN，拿去识别医学影像（只需要少量医学图片微调）。

### 知识点 13：更多学习范式一览

| 范式 | 一句话说清 |
| :---- | :--------- |
| **自监督学习** 🪞 | 从数据本身构造监督信号（如 BERT 预测被遮住的词） |
| **度量学习** 📏 | 学一个距离函数，让同类样本距离近，异类远 |
| **对比学习** 🔁 | 让相似样本的表示靠近，不相似的远离 |
| **对抗学习** 🎭 | 生成对抗样本攻击模型，再让模型学会抵抗 |
| **元学习** 🧠 | "学会学习"——在新任务上快速适应 |

## 六、📚 学习理论要点

### 知识点 14：没有免费午餐定理（No Free Lunch Theorem）🍽️

> **核心结论**：没有一种算法在所有问题上都是最好的。

```text
算法A在问题集P₁上表现好
算法B在问题集P₂上表现好

如果对所有可能的问题求平均，所有算法的期望性能是一样的。
```

**实际意义**：选择算法需要根据**具体问题**判断，不能盲目迷信某种算法。

### 知识点 15：丑小鸭定理（Ugly Duckling Theorem）🦆

> **核心结论**：没有绝对客观的分类标准，分类依赖于我们的**主观偏好**。

```text
丑小鸭和白天鹅的"差异"，取决于你用什么标准来衡量：
- 按外形分：丑小鸭 ≠ 白天鹅
- 按基因分：丑小鸭 = 白天鹅（都是天鹅科）
```

### 知识点 16：奥卡姆剃刀（Occam's Razor）🪒

**Entities should not be multiplied unnecessarily.** —— William of Occam

> **核心结论**：在效果相同时，**选择最简单的那个模型**。

```text
如果有两个模型在训练集上表现一样好：
  模型A：3 个参数（简单） 👍
  模型B：30 个参数（复杂）

奥卡姆剃刀说：选模型A！更简单的模型更可能泛化到新数据。
```

### 知识点 17：归纳偏置（Inductive Bias）🎯

> **核心问题**：数据是有限的，但可能的模型是无限的。如何选择？

**归纳偏置** = 学习算法在假设空间中做选择时的**先验偏好**。

| 算法 | 归纳偏置 |
| :---- | :---- |
| **线性回归** 📈 | 假设 x 和 y 的关系是线性的 |
| **CNN** 🧠 | 假设图像特征具有局部性和平移不变性 |
| **RNN** 🔄 | 假设数据具有时序依赖 |
| **决策树** 🌳 | 倾向于选择较短的树 |

> 💡 **没有归纳偏置就无法学习**——数据本身不足以确定唯一模型，必须加上先验假设。
