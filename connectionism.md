## 一、连接学派简介

### 1. 生物启发
- **生物视觉信息处理**：
  - 腹侧通路（V1、V2、V4）：处理形状、颜色等信息
  - 背侧通路（V1、V2、V5）：处理运动等信息
- **生物神经元**：
  - 人脑约有 10¹¹ 个神经元，每个神经元与 10³~10⁴ 个其他神经元相连
  - **工作状态**：
    - 兴奋：细胞膜电位 > 动作电位阈值 → 神经冲动
    - 抑制：细胞膜电位 < 阈值
  - **学习与遗忘**：突触的传递作用可增强或减弱（可塑性）

### 2. 连接主义（Connectionism）核心思想
- 又称仿生学派 / 生理学派，源于仿生学
- 高级智能行为是从大量神经网络的连接中**自发涌现**
- 主要原理：神经网络结构 + 连接机制 + 学习算法

### 3. 连接学派发展历程（三起两落）
| 时期 | 标志事件 |
|------|---------|
| 第一次兴起（1940s-60s） | MP模型、感知器 |
| 第一次低谷（70s） | XOR问题、AI寒冬 |
| 第二次兴起（80s-90s初） | Hopfield网络、BP算法 |
| 第二次低谷（90s中-2000s） | 维度灾难、SVM崛起 |
| 第三次兴起（2006-至今） | 深度学习、AlexNet |

## 二、神经网络基本概念与近代史

### 1. 人工神经网络发展简史时间线
| 年份 | 事件 |
|------|------|
| 1943 | McCulloch-Pitts MP 模型 |
| 1958 | Rosenblatt 感知器模型 |
| 1960 | Widrow & Hoff ADALINE |
| 1969 | Minsky-Papert 《Perceptrons》——XOR问题 |
| 1982 | Hopfield 网络（HNN） |
| 1986 | Rumelhart-Hinton-Williams BP算法 |
| 1995 | Vapnik SVM |
| 2006 | Hinton《Science》深度神经网络（DBN） |
| 2012 | AlexNet 赢得 ImageNet ILSVRC |

### 2. MP 模型（McCulloch-Pitts, 1943）
- **思想**：简化生物神经元，建模人工神经元
- **特点**：
  - 多输入单输出
  - 输入分兴奋性 / 抑制性
  - 具备空间整合特性和阈值特性
  - **布尔输入和输出**（0或1）
  - **阶跃响应函数**
  - **没有可学习的权重**
- **意义**：验证型模型——证明神经元可以做计算
- **可实现**：逻辑与、逻辑或

### 3. 感知器模型（Perceptron, Rosenblatt 1958）
- **特点**：
  - **实值的输入和输出**（区别于 MP 的布尔）
  - 阶跃响应函数
  - **有可学习的权重**
- **几何解释**：通过一个（超）平面将输入空间划分为两个半空间
- **限制**：
  - 属于线性模型，**无法解决线性不可分问题**（如 XOR）
  - 即使少量不可分样本也会导致学习不收敛

### 4. ADALINE 模型（Widrow & Hoff, 1960）
- **全称**：ADAptive LInear NEuron
- **特点**：
  - 激活函数：由阶跃函数变为**连续函数**
  - 增加了 **Quantizer（量化函数）** 用于类别预测
  - 定义了**代价函数**概念（均方误差 MSE）
  - 使用**梯度下降**优化权重
- **意义**：是多层网络中 BP 算法的先驱

### 5. XOR 问题与第一次 AI 寒冬
- 1969 年 Minsky & Papert 在《Perceptrons》中指出：2 层感知器无法解决 XOR
- XOR 问题**不是线性可分**的
- **解决方法**：用三层感知器可以解决
- **AI 寒冬原因**：
  - 机器翻译进展熄火
  - 《Perceptrons》对感知器的批判
  - Lighthill report（1973）批评 AI

### 6. Hopfield 神经网络（HNN, 1982）
- AI 寒冬后的复兴
- 1987 年贝尔实验室基于 HNN 研制神经网络芯片
- **能力**：联想记忆、神经优化
- **结构**：递归 / 反馈神经网络（输出到输入有反馈连接）
- 每个吸引子（稳定状态）储存一个模式（pattern）

### 7. BP 神经网络（1986）
- 也称多层感知器（MLP）
- **历史**：BP 算法实际由 Paul Werbos 1974 年博士论文首次论证；1986 年 Rumelhart-Hinton-Williams 在 Nature 上发表使其广为人知
- **意义**：解决了多层网络的训练问题
- **万能近似定理**（George Cybenko, 1989）：多层前馈网络可近似任意函数
- **存在的问题**：
  - **维度灾难**：全连接前馈网络参数空间维数大幅增加
  - **泛化问题**：高维空间多峰非线性优化难以找到全局最优
  - 在 1990s 算力有限的情况下难以扩展

### 8. SVM 的崛起（1995, Vapnik）
- 利用**核函数**处理非线性可分问题（神经网络则用隐藏层变换）
- 优势：无需调参、速度快、全局最优解
- 标志神经网络第二次进入低谷
- **趣事**：
  - 神经网络低谷时改名"并行分布式处理"（Parallel Distributed Processing）
  - SVM 在 1995 年复兴时取名"Support Vector Network"蹭热点

### 9. 深度学习时代的开启
- **2006 年 Hinton 在 Science 上发文**：用 **DBN（深度信念网络）** 在未标注数据上预训练深度神经网络
- DBN 是首批成功应用深度架构训练的非卷积模型，MNIST 上超越 SVM
- 标志性事件：
  - 2006 手写字符识别 MNIST
  - 2010 语音识别
  - **2012 AlexNet** ImageNet ILSVRC 冠军（8 层、Top5 error 16.4%）
  - 2016 AlphaGo 击败李世石
- **成功原因**：
  - 算法哲学：模块化 + 层次化、拓扑叠加、分而治之
  - 算法细节：ReLU、Dropout、Pooling
  - 工程成本：深度学习工具大量出现，降低门槛
- **存在的问题**：
  - 暴力美学：拼数据、拼资源
  - "炼丹"：可解释性差
  - 与符号主义的融合（先验知识表达和融入）

## 三、神经网络基本思想与结构

### A. 前馈神经网络（近似函数）

#### 1. 基本概念
- 又称多层感知器（MLP）
- **目标**：通过映射 y = f(x; θ)，调整 θ 去逼近函数 f*
- **关键创新**：增加了**非线性激活单元**，使网络拥有非线性建模能力

#### 2. 网络结构
- **三种网络层**：
  - 输入层（含偏置单元）
  - 隐含层（个数和神经元数为超参数，含偏置单元）
  - 输出层
- **连接特性**：
  - 相邻层全连接（也称全连接层）
  - 同层无连接
  - 单向连接（信息可双向传播）

#### 3. 非线性激活函数
- **Logistic（Sigmoid）**：输出恒 > 0
- **Tanh**：零中心化
- **作用**：使网络拥有非线性建模能力

#### 4. 前向传播 vs 后向传播
| | 前向传播 | 后向传播 |
|---|----------|----------|
| 方向 | 输入层 → 输出层 | 输出层 → 输入层 |
| 用途 | 计算输出 | 更新参数 |
| 核心 | 加权求和 + 激活 | **链式法则** |
| 驱动 | 输入 | **损失函数**（如均方误差） |

#### 5. 通用近似定理
- 多层前馈网络可以近似任意函数
- 两种方式：
  - 增多层数 / 减少隐层神经元
  - 只需三层 / 隐层神经元个数多

#### 6. 前馈神经网络特点
- **优点**：
  - 强大逼近特性、通用函数拟合
  - 较好的容错性
- **缺点**：
  - 收敛速度慢
  - 可能陷入局部极值
  - 没有选择隐层数和节点数的通用准则

### B. Hopfield 网络（解决优化问题）

#### 1. HNN 概述
- 单层、全连接、固定参数的反馈型神经网络
- 输出到输入均有反馈连接
- 每个神经元既是输入也是输出
- 引入**能量函数**判断网络稳定性
- 分两种：
  - **离散型（DHNN）**：联想记忆
  - **连续型（CHNN）**：优化问题

#### 2. 离散 Hopfield 网络（DHNN）
- 二值神经元：{0,1} 或 {-1,1}（抑制/激活）
- 各节点**没有自反馈**
- **两种工作方式**：
  - 异步（串行）：t 时刻只有一个神经元状态变化
  - 同步（并行）：所有神经元同时变化
- **能量函数**：
  $$E = -\frac{1}{2}\sum_{i=1}^{N}\sum_{j=1,j\neq i}^{N}w_{ij}v_iv_j + \sum_{i=1}^{N}\theta_i v_i$$
- 系统运行中能量不断减少，最终达到极小值（稳定态）
- W 是对角元为 0 的对称矩阵 → 串行稳定性
- **应用**：联想记忆（但存在**串扰**问题）

#### 3. 连续 Hopfield 网络（CHNN）
- 拓扑结构与 DHNN 一致
- **更接近生物神经元**
- 神经元状态取 0~1 连续值
- 激活函数：**S 型连续函数**（非阶跃）
- 神经元同步工作
- 稳定条件：传递函数连续有界 + 权重矩阵对称
- **应用**：1985 年 Hopfield & Tank 求解 TSP 问题
  - 思路：将代价函数写为 Hopfield 能量函数形式 → 平衡态即为问题的解

#### 4. 玻尔兹曼机（Boltzmann Machine）
- 解决 Hopfield 串扰和局部最优问题
- 通过**概率分布**让单元状态变化，避免陷入局部最优
- 保留 Hopfield 假设：权重对称、自身无连接、二值输出
- **新增**：可见单元 + 隐藏单元（隐变量带来更强表达能力）

#### 5. 受限玻尔兹曼机（RBM）
- 由可见层和隐藏层构成
- **层内无连接，层间全连接**
- 信息双向流动
- **学习目标**：最大化似然
- 解决了含隐变量的 BM 训练困难问题

#### 6. 深度玻尔兹曼机（DBM）
- 多个 RBM 堆叠 → 隐藏层变多
- **训练方式**：贪婪、逐层、无监督预训练
- 适用于**无监督学习**（无向连接）

#### 7. 深度信念网络（DBN）
- 靠近可视层用贝叶斯信念网络（**有向**），最远端用 RBM（**无向**）
- **是一种生成模型**，深层概率有向图模型
- **训练**：逐层无监督预训练 + 标注数据精调
- 适用于**监督学习**
- **意义**：2006 年 Hinton 在 Science 上证明深度架构有效（MNIST 上超 SVM）
- **应用**：识别特征、分类数据、生成数据（如生成手写数字）

### C. 循环神经网络 RNN（近似动力学系统）

#### 1. 序列化数据
- 时序信号特点：上下文强关联、序列长度变化
- CNN 难以直接建模
- 典型：音频、视频、文本、生物信号

#### 2. 序列建模需求
- 处理变长序列
- 保留序列顺序
- 建模上下文依赖
- 整个序列共享模型参数

#### 3. RNN 简史
- 1986 Jordan Network
- 1990 Elman Network
- 1994 RNN 概念首次提出
- 1997 LSTM（Schmidhuber）
- 2010 RNN 用于语言模型
- 2009-2012 Schmidhuber 团队多次获得国际竞赛冠军

#### 4. RNN 基本结构
- 三种网络层：输入层、隐含层、输出层
- 两种表示：折叠表示、展开表示
- 隐藏状态：
  $$s_t = f(s_{t-1}, x_t)$$
- 特征 sₜ 不仅取决于当前输入 xₜ，还取决于 sₜ₋₁

#### 5. RNN 与非线性动力系统
- RNN 可理解为通用的非线性系统
- **通用近似定理**：完全连接的 RNN 可近似任何非线性动力系统

#### 6. RNN 典型应用
| 应用 | 类型 |
|------|------|
| 情感分类 | 序列 → 类别 |
| 中文分词 | 同步序列 → 序列（B/M/E/S 标注） |
| 信息抽取 | 同步序列 → 序列（B/I/O 标注） |
| 机器翻译 | 异步序列 → 序列 |
| 图像描述（Image Caption） | 图像 → 序列 |
| 手写体识别 | 序列 → 序列 |
| 计算机合成音乐 | 生成 |

#### 7. RNN 特点
- **优点**：
  - 处理任意长度序列
  - **参数共享**（不同时间步用相同权重）
  - 通用计算能力（按任意精度逼近非线性系统）
- **缺点**：
  - **建模单向性**：输出仅与 t 时刻之前输入有关
  - **长序列记忆丢失（梯度消失）**

#### 8. 梯度消失/爆炸的原因
- 反向传播时，根据链式法则，激活函数求导值需要**连乘**
- 不同时刻参数共享，导致重复对相同矩阵 W 求 t 次幂
- 通过特征值分解 W = V·diag(λ)·V⁻¹：
  - W^t = V·diag(λ^t)·V⁻¹
  - λ > 1 → 梯度爆炸（少见）
  - λ < 1 → 梯度消失（**大部分情况**）
- BP 算法推广为 BPTT（Back Propagation Through Time）

#### 9. 改善梯度消失的途径
- 改进 RNN 结构（**LSTM、GRU**）
- 使用合适的激活函数（如 tanh）

### D. 长短期记忆网络（LSTM, 1997）

#### 1. 核心贡献
- 引入**自循环**，产生梯度长时间持续流动的路径（gradient path）
- 解决梯度消失的根本原因

#### 2. 状态设计
- **细胞状态 C**：长记忆
- **隐藏状态 h**：短记忆

#### 3. 三个门控机制（gate）
| 门 | 作用 |
|----|------|
| **遗忘门**（forget gate, fₜ） | 控制自循环权重，对上一节点输入选择性忘记 |
| **输入门**（input gate, iₜ） | 选择性记忆当前阶段输入 |
| **输出门**（output gate, oₜ） | 控制 LSTM 单元的输出 |

#### 4. 输入输出
- 输入：cₜ₋₁、hₜ₋₁、xₜ
- 输出：cₜ、hₜ、yₜ

#### 5. 计算流程
1. **遗忘门**：fₜ = σ(W_f·[hₜ₋₁, xₜ] + b_f)
2. **输入门**：
   - iₜ = σ(W_i·[hₜ₋₁, xₜ] + b_i)
   - C̃ₜ = tanh(W_C·[hₜ₋₁, xₜ] + b_C)
3. **细胞状态更新**：cₜ = fₜ ⊗ cₜ₋₁ + iₜ ⊗ C̃ₜ
   - 有选择地忘记 + 有选择地记忆
4. **输出门**：
   - oₜ = σ(W_o·[hₜ₋₁, xₜ] + b_o)
   - hₜ = oₜ ⊗ tanh(cₜ)

#### 6. LSTM 与 RNN 对比
| | 联系 | 区别 |
|---|------|------|
| LSTM vs RNN | 都是循环链式结构；都假设单向依赖 | LSTM 引入大量非线性单元；增加细胞状态 C 和门控；选择性建模长期记忆 |

---

### E. 双向 RNN
- 时刻 t 输出**同时取决于过去和未来**信息
- 每个隐藏层有两个节点：
  - 一个正向计算
  - 一个反向计算
- 输出层由两值决定
- **正向和反向计算的权重不共享**

## 四、关键考点小结

### 1. 各神经元/网络模型对比
| 模型 | 输入输出 | 激活 | 权重学习 | 主要贡献 |
|------|---------|------|----------|---------|
| MP (1943) | 布尔 | 阶跃 | 否 | 验证模型可计算 |
| Perceptron (1958) | 实值 | 阶跃 | 是 | 引入可学权重 |
| ADALINE (1960) | 实值 | 连续 | 梯度下降 | MSE代价函数、梯度下降 |
| BP-MLP (1986) | 实值 | 非线性（Sigmoid/Tanh） | BP算法 | 解决多层训练 |
| Hopfield | 二值/连续 | 阶跃/S型 | 固定 | 联想记忆/优化 |
| RBM | 二值 | - | 最大似然 | 无监督预训练 |
| RNN | 序列 | tanh | BPTT | 序列建模 |
| LSTM | 序列 | tanh+门控 | BPTT | 解决梯度消失 |

### 2. 三次兴起 & 两次低谷的根本原因
- 第一次低谷：**线性不可分问题**（XOR）
- 第二次低谷：**算力和模型不足 + SVM 竞争**
- 第三次兴起：**深度学习 + 大数据 + GPU 算力**

### 3. 重要公式
- **Hopfield 能量函数**：E = -½ΣΣwᵢⱼvᵢvⱼ + Σθᵢvᵢ
- **LSTM 细胞更新**：cₜ = fₜ⊗cₜ₋₁ + iₜ⊗C̃ₜ
- **RNN 隐状态**：sₜ = f(sₜ₋₁, xₜ)

### 4. 三个基本架构对比
| 架构 | 解决问题 | 拓扑结构 |
|------|---------|---------|
| 前馈网络 | 函数近似 | 单向、分层 |
| Hopfield | 优化问题、联想记忆 | 全连接反馈 |
| RNN | 动力学系统 / 序列建模 | 循环（带时序） |
